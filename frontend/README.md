# 🚒 AUTONOME_APP  
### Application officielle du **Syndicat AUTONOME SDMIS**

Bienvenue dans le dépôt de l’application mobile et web du **Syndicat AUTONOME SDMIS**, conçue pour offrir aux agents SPP et PATS un accès simple, rapide et moderne à l’information syndicale, aux outils pratiques et aux services internes.

---

## 🟨 Objectifs de l’application

- Donner un accès direct aux informations syndicales  
- Faciliter la communication entre le syndicat et les agents  
- Proposer des outils pratiques (alertes, contacts, sondages, documents, etc.)  
- Moderniser l’expérience numérique du syndicat  
- Centraliser les ressources dans une interface claire et intuitive  

---

## 🟩 Fonctionnalités principales

### 📱 **Frontend (React Native / Expo)**
- Navigation par onglets (Home, Alerte, Contact, Sondages…)  
- Interface moderne aux couleurs du syndicat  
- Composants optimisés pour mobile  
- Gestion des états, typage TypeScript, composants réutilisables  
- Intégration future : notifications, API backend, authentification

### 🖥️ **Backend (Python / FastAPI ou équivalent)**
- Architecture prête pour les endpoints API  
- Sécurisation progressive  
- Gestion des données syndicales  
- Connexion future à une base de données
